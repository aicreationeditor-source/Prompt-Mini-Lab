STEP 1 — 원본 통합

CSV·XLSX 여러 개를 떨구면 헤더 기준으로 세로 통합한다. 원본 추적용 _출처 열이 자동으로 붙는다. 원본은 절대 수정되지 않는다 — 네 VBA의 "_통합데이터 이후 수정 금지" 원칙 그대로.

파일을 여기로 끌어오거나 클릭해서 선택 (CSV / XLSX, 복수 가능)

STEP 2 — 전처리 규칙

규칙은 위에서 아래로 순서대로 적용된다. 각 규칙 옆에 통과/제외 건수가 찍혀서 어느 규칙이 데이터를 깎아먹는지 바로 보인다.

STEP 3 — 보고서 설정

네 VBA의 _차트설정 시트에 해당한다. 그룹 기준·집계·차트 유형을 정하면 표+차트가 생성된다.

설정 보존 — 재사용이 "급"의 핵심

전처리 규칙 + 보고서 설정을 JSON으로 내보내고, 다음에 새 데이터에 그대로 불러온다. 네 VBA에서 설정 시트가 하던 역할.
